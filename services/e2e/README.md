# E2E test service

This service is an e2e test service, to help us ensure that Nuon is working as expected.

Please read more in the [E2E service wiki](../../wiki/e2e-service).

## Directories

* `nuon` - the terraform to manage the nuon deployment of this application
* `api` - an introspection api that exposes details about the install when running
* `worker` - a basic background worker
* `terraform` - terraform resources
