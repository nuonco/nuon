# AWS packages

We maintain different client libraries that build on the aws-sdk here.

* `aws-assume-role` - assume an IAM role
* `ecr-authorization` - fetch credentials from an ecr repo
* `eks-client` - authenticate with `eks`
* `s3downloader` - utilities for fetching data from s3
* `s3uploader` - utilities for storing data in s3
* `credentials` - an opinionated library for fetching/caching/reusing credentials
