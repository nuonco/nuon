Nuon supports SDKs for common languages, which can be used to integrate Nuon directly into your product.

## Supported SDKs

<CardGroup cols={2}>
  <Card title="Go" icon="golang" href="https://github.com/nuonco/nuon-go">
  `nuon-go` sdk for Go applications.
  </Card>

  <Card title="Python" icon="python" href="https://github.com/nuonco/nuon-python">
  `nuon` sdk for Python applications.
  </Card>

  <Card title="Elixir" icon="code" href="https://github.com/nuonco/nuon-elixir">
  `nuon` sdk for Elixir applications.
  </Card>
</CardGroup>

## Integration

For additional information on integrating Nuon into your product, refer to the [integration guide](/guides/sdk-integration).

## OpenAPI Support

Both an OpenAPI v2 and OpenAPI v3 spec are published via our API, and can be used to generate additional clients.

* [OpenAPI v2 spec](https://api.nuon.co/oapi/v2)
* [OpenAPI v3 spec](https://api.nuon.co/oapi/v3)

<Note>
We plan on supporting SDKs for most popular languages, as our users need them.

If you would like to use a currently unsupported language, please [get in touch](mailto:team@nuon.co) so we can consider
  adding it to our list of first class SDKs.
</Note>
