## Setup

The Terraform provider must be provided with an API token, and an org ID.

To set your `api_token`:
1. Fetch an api token using `nuon api-token`.
1. Export `NUON_API_TOKEN=token` _or_ use the `api_token` variable on the provider.

<CodeGroup>
```hcl providers.tf
provider "nuon" {
  api_token = "your-api-token"
}
```
```hcl versions.tf
terraform {
  required_version = ">= 1.3.7"

  required_providers {
    nuon = {
      source  = "nuonco/nuon"
      version = ">= 0.8.0"
    }
  }
}
```
</CodeGroup>
