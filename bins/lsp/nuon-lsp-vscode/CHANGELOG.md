# Change Log

All notable changes to the "Nuon LSP" extension will be documented in this file.

## [0.0.2] - 2024-12-17

### Added
- Extension icon and branding with Nuon logo
- Comprehensive README with features, usage, and troubleshooting
- CHANGELOG for version tracking
- MIT LICENSE file
- Enhanced package.json metadata:
  - Categories (Programming Languages, Linters, Formatters)
  - Keywords for marketplace discoverability
  - Repository and bug tracking links
  - Homepage link

### Improved
- Better description in marketplace listing
- Professional presentation with icon and documentation
- .vscodeignore for cleaner package

## [0.0.1] - 2024-12-17

### Added
- Initial release of Nuon LSP extension
- Language server support for TOML files
- Real-time syntax validation
- Auto-completion support
- Hover documentation
- Go to definition
- Configurable server path
- Development mode with TCP connection support
- Automatic server discovery in PATH

### Features
- Stdio communication mode for production use
- TCP mode for local development
- Automatic server binary detection
- Clear error diagnostics and messaging
- VS Code Output channel for debugging
